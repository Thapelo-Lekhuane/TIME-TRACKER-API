# Time Tracker Frontend

React + TypeScript + Vite frontend for the Time Tracker API.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The frontend will run on http://localhost:5173 and proxy API requests to http://localhost:3000.

## Build

```bash
npm run build
```

## Features

- Employee dashboard with clock in/out functionality
- Manager dashboard for attendance tracking and leave approvals
- Admin dashboard for user and campaign management
- Role-based routing and authentication
- Clean grey/blue/white UI design
